# 📊 Retail Inventory Profit Analysis

## Project Overview

This is an End-to-End Data Analyst project developed using:

- Microsoft Excel
- MySQL
- Power BI
- Python
- Streamlit

The project analyzes retail inventory data to identify sales performance, supplier profit, product performance, and monthly sales trends.

---

## Objectives

- Analyze Sales, Cost and Profit
- Identify Top Selling Products
- Identify Top Suppliers
- Analyze Monthly Sales
- Analyze Category-wise Sales
- Build an Interactive Dashboard

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Streamlit
- MySQL
- Excel
- Power BI

## Features

- Upload Excel File

## How to Run

### Clone Repository

```
git clone <your-github-link>
```

### Install Requirements

```
pip install -r requirements.txt
```

### Run Streamlit

```
streamlit run app.py
```
